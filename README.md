
# Legalix AI Grant Writing Assistant

<img width="1536" height="1024" alt="legalix-cover" src="https://github.com/user-attachments/assets/c17dbb77-33df-4c3c-977e-2001dcd0437a" />

This AI project showcases Legalix, an executive-ready grant writing and grant-readiness workflow for nonprofit teams that need stronger proposals, clearer funder alignment, reusable grant assets, and review-ready funding-fit signals.

## Project Purpose

Legalix demonstrates an AI-assisted grant operations experience for nonprofit teams. The showcase focuses on proposal intelligence, funder alignment, evidence strength, budget clarity, narrative consistency, and reusable grant-writing assets.

## Key Features

- Executive Legalix cover page
- Legalix logo and brand system section
- Grant readiness console
- Interactive readiness sliders for funder alignment, evidence strength, budget clarity, and narrative voice
- Live funding-fit score and recommendation summary
- Animated grant document intelligence preview
- Responsive layout for desktop and mobile browsers
- Static site architecture with no backend requirement

## Live Demo Sections

- Cover Page: Branded Legalix executive visual
- Brand System: Logo-based visual identity and design direction
- Live Product Demo: Grant readiness console
- Executive Value: Proposal intelligence, reusable assets, and review confidence

## Technology Stack

- HTML
- CSS
- Vanilla JavaScript
- Canvas-based document intelligence animation
- Static assets stored locally in `assets/`

No framework, build step, server runtime, or database is required.

## Run Locally

Open `index.html` directly in a browser, or run a local static server:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000/
```

## Deploy With GitHub Pages

1. Add the Legalix files to the GitHub repository.
2. Make sure `index.html` is at the repository root.
3. Commit and push the files.
4. In GitHub, open repository settings.
5. Go to Pages.
6. Choose the branch and root folder to publish.
7. Save the Pages settings.

The published project URL should look like:

```text
https://ldodds-dev.github.io/legalix.github.io/
```
